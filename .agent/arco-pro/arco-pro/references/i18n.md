# 国际化规范

## 目录结构

```
locale/
├── index.ts              # 主入口，配置 i18n
├── zh-CN.ts             # 中文全局翻译（合并所有模块）
├── en-US.ts             # 英文全局翻译
└── [lang]/              # 可选：语言包目录

views/[module]/locale/
├── zh-CN.ts             # 模块中文翻译
└── en-US.ts             # 模块英文翻译
```

## 翻译 Key 命名规范

### 命名格式

| 类别 | 格式 | 示例 |
|------|------|------|
| 菜单 | `menu.module[.action]` | `menu.users`, `menu.users.list` |
| 表格列 | `module.columns.field` | `users.columns.username` |
| 表单字段 | `module.form.field` | `users.form.username` |
| 表单标签 | `module.form.fieldLabel` | `users.form.usernameLabel` |
| 操作按钮 | `module.operation.action` | `users.operation.search` |
| 消息提示 | `module.message.action` | `users.message.createSuccess` |
| 状态值 | `module.status.value` | `users.status.active` |
| 模态框 | `module.modal.type` | `users.modal.deleteTitle` |
| 占位符 | `module.placeholder.field` | `users.placeholder.username` |
| 验证消息 | `module.rules.ruleName` | `users.rules.required` |

## 组件中使用

### 模板中使用

```vue
<template>
  <!-- 使用 $t() -->
  <a-card :title="$t('menu.users.list')">
    <a-button>{{ $t('users.operation.search') }}</a-button>
  </a-card>

  <!-- 面包屑 -->
  <Breadcrumb :items="['menu.users', 'menu.users.list']" />
</template>
```

### Script 中使用

```typescript
import { useI18n } from 'vue-i18n';

const { t } = useI18n();

// 直接调用
const message = t('users.message.createSuccess');

// 带参数
const msg = t('users.message.deleteBatchConfirm', { count: selectedCount });

// 在计算属性中
const columns = computed(() => [
  { title: t('users.columns.username'), dataIndex: 'username' },
]);

// 动态状态
const getStatusText = (status: string) => t(`users.status.${status}`);
```

```typescript
  { label: t('users.role.admin'), value: 'admin' },

const statusOptions = computed(() => [

  { label: t('users.status.inactive'), value: 'inactive' },

```

### 表格列动态翻译

```typescript
const columns = computed<TableColumnData[]>(() => [
  {
    title: t('users.columns.status'),
    dataIndex: 'status',
    render: ({ record }) => t(`users.status.${record.status}`),
  },
]);
```
